# DeepLearning – Textbook Companion Repository

This repository contains Python and TensorFlow notebook programs designed as a companion resource for Deep Learning textbooks and classroom teaching.

All notebooks focus on clear explanation, step-by-step execution, and practical understanding of deep learning concepts using simple and well-structured code examples.

This repository is developed mainly for undergraduate and postgraduate students, educators, and self-learners.

---

## Purpose

- Support theoretical learning through practical experiments  
- Provide ready-to-run Jupyter and Google Colab notebooks  
- Enable students to visualize and understand deep learning concepts  
- Act as a permanent reference resource for textbook-based learning  

---

## Repository Structure

The repository is organized unit-wise and topic-wise:
DeepLearning/
│
├── Unit-01_Introduction/
├── Unit-02_Deep_Neural_Networks/
├── Unit-03_Convolutional_Neural_Networks/
├── Unit-04_Recurrent_Neural_Networks/
├── Unit-05_Advanced_Topics/


Each folder contains:

- Concept explanation notebooks  
- Step-by-step experiments  
- Visualization programs  
- Practice exercises

## How to Use

You can open any notebook directly in Google Colab using the link format:

https://colab.research.google.com/github/suneelcs1615/DeepLearning/blob/main/path_to_notebook.ipynb


To create your own editable copy:

- Click **File → Save a copy in Drive** in Colab  
- Or change the URL ending from `/edit` to `/copy`  

---

## Suggested Learning Flow

1. Read the theory from the textbook  
2. Open the related notebook from this repository  
3. Run each code cell step by step  
4. Observe outputs and visualizations  
5. Perform suggested exercises
